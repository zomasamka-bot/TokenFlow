# TokenFlow

Accept. Execute. Proof.

Execution and proof layer for redeemable digital rights.

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the app.

## Build

```bash
pnpm build
pnpm start
```

## Features

- **Accept** - View available digital rights
- **Execute** - Redeem rights with one tap
- **Proof** - Get instant, verifiable proof of execution
- **Activity** - Track execution history

## Tech Stack

- Next.js 15 (App Router)
- React 19
- Tailwind CSS
- TypeScript
